Export & Download

You can download most information on the respective pages directly:

  • You can download the cuticular collagens, directly from the webpages reacahble under "genes"
  • You can download the sequences of all collagens of a particular class using the "export" function on the respective pages for the clusters/classes. (TIP: If you want to download all cuticular collagens, use the export function on this page )
  • On the pages describing individual cluster/classes you'll find a table which easily exports to Excel, with many interesting paramters.
  • If you need anything else just contact us!
